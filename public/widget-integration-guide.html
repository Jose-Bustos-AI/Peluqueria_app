🎯 Widget de Reservas Pleno - Guía de Integración

📋 Integración Básica

Para integrar el widget en tu sitio web, simplemente añade este código antes del cierre del tag </body>:

<!-- Widget de Reservas Pleno -->
<script src="https://tu-dominio.com/widget-embed.v2.js"></script>
<script>
  // Inicializar widget con configuración básica
  initPlenoWidget({
    companyName: 'Tu Centro de Bienestar',
    companyAddress: 'Tu dirección completa',
    bookingUrl: 'https://tu-dominio.com/widget'
  });
</script>

⚙️ Configuración Avanzada

Propiedad Tipo Por Defecto Descripción
backgroundColor string 'rgba(37, 44, 88, 1)' Color de fondo del widget
primaryColor string '#ef4444' Color principal (botones, headers)
textColor string '#ffffff' Color del texto
width string '384px' Ancho del widget
height string '100vh' Altura del widget
position string 'fixed' Posición CSS (fixed, absolute)
right string '20px' Distancia desde la derecha
top string '20px' Distancia desde arriba
autoOpen boolean false Abrir automáticamente al cargar
showCloseButton boolean true Mostrar botón de cerrar
triggerSelector string null Selector CSS para elementos que abren el widget
companyName string 'Centro Pleno' Nombre de tu empresa
companyAddress string Dirección por defecto Dirección de tu centro
bookingUrl string URL actual + '/widget' URL de tu página de reservas

<!-- En tu HTML -->

<a href="#" class="link-reserva">Pedir cita aquí</a>

Widget Responsivo

initPlenoWidget({
  width: window.innerWidth < 768 ? '100vw' : '400px',
  height: window.innerWidth < 768 ? '100vh' : '600px',
  right: window.innerWidth < 768 ? '0' : '20px',
  top: window.innerWidth < 768 ? '0' : '20px'
});

🔧 API de Control

Una vez inicializado, puedes controlar el widget mediante JavaScript:

// Abrir el widget
PlenoWidget.open();

// Cerrar el widget
PlenoWidget.close();

// Alternar visibilidad
PlenoWidget.toggle();

// Verificar si está abierto
if (PlenoWidget.isOpen()) {
  console.log('Widget está abierto');
}

// Destruir completamente el widget
PlenoWidget.destroy();

⚠️ Consideraciones Importantes

  • HTTPS: El widget debe servirse desde HTTPS para funcionar correctamente
  • CORS: Asegúrate de que tu servidor permita el embedding del iframe
  • Responsive: El widget se adapta automáticamente a pantallas móviles
  • Performance: El script es ligero (~15KB) y no afecta la carga de tu página
  • Privacidad: Cumple con GDPR - no almacena cookies de terceros

🔐 Configuración del Servidor

En tu servidor, asegúrate de permitir el embedding:

# En .htaccess (Apache)
Header always append X-Frame-Options SAMEORIGIN

# En nginx.conf
add_header X-Frame-Options SAMEORIGIN;

# O para permitir dominios específicos:
# add_header X-Frame-Options "ALLOW-FROM https://cliente.com";

📱 Soporte Móvil

El widget incluye soporte automático para dispositivos móviles:

  • Se adapta a pantalla completa en móviles (< 480px)
  • Gestos táctiles optimizados
  • Botón flotante responsivo
  • Carga rápida en conexiones lentas
